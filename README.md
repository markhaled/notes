## Notes

A modern Android note-taking application built in with Jetpack Compose that enables users to create,organize,edit and manage notes 

 ## Tech Stack
 
 | Technology              |     Purpose |
 |-------------------------|-------------|
 | kotlin | Programming Language |
 | MVVM | Architecture |
 | Jetpack Compose | UI |
 | Hilt | Dependency injection |
 | Room | Local Database |
 | Retrofit | API Communication |
 | Navigation Compose | Navigation |
 | Coroutiens | Asynchronous Programming |
 | Flow | Reactive UI |
